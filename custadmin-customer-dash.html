Customer

A customer is an entity that engages in transactions for services and products with the service provider.

A customer is created using a Party (represents the person/individual) that is ideally created before you create a customer, so you select an existing party when you create the customer. The option to create the party is presented during the creation of the customer if the required party is not already created in the system.

Customer search

Click the Customer icon to open the Customer page and search for existing customers.

  • Click the advanced search icon to use the advanced search box.
Note: On the dashboard, use the search box and search category filters to search for a customer.
Use the options under Actions to view, edit, or change status of selected customers.
