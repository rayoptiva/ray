Add Account Contact

Add a contact for this account.

  1. Click the Contacts tab on the Create Account page.
  2. Click Add or + Add Contact.
  3. Select a Medium Type.
  4. Complete other relevant fields.
    Note: Each contact should have name and valid phone number.
  5. [Optional] To make this Medium Type the preferred option for this account, you must select the Preferred checkbox.
  6. Click Save.
Note: Click Cancel to delete this record, or save in draft form so that you can return to this at a later time.
