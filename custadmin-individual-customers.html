Creating Customers

Create a customer by selecting party and adding other relevant information

Each customer is associated with a party (individual) which you can add or select by using the search option when you create the customer.

The Create Customer screen is accessible in the following ways:

  • Click Customer > + Create > Individual Create Customer screen.
  • From Home > +Create > New Customer screen, click + Create to open the Create Customer screen.

Click each of the following entities, add the relevant details and submit at any time after the required sections are completed.

  • Customer [required]
  • Contacts
  • Credit Profile

The Submit button becomes available to create the customer record when the required fields are populated.

The Save button enables you to save your current inputs as a draft to work on at another time.

Customer

Add a new Customer by selecting a Party and adding validation dates.

  1. To create a customer, you must assign a party by either of the following methods:

    • Select an existing party through the Select Party search box.

      You can search by Party ID, Given Name, Family Name, or Full Name.

      Select the party that you want to use from the search results and the relevant information for that party will populate automatically.

    • Click + Create Party to create a new party.
      Note: A Party must be validated to be used for customer creation.
  2. Add the Valid from and Valid until dates.

    This is the validity period for this customer.

  3. Click Save to save as draft or click Submit to create this customer record.

    The Submit button becomes active when all the required fields are completed and a summary screen is presented. When you submit, you have the option to edit sections on the summary screen and submit again, or click Confirm to create the Customer.

    Note: Click Cancel to delete this record, or save as draft so that you can return to this at a later time.
  4. Continue to the next section.

Contacts

This section is pre-populated from the Party selection but you can add another contact if required by using the following instruction.

Note: You can't edit contact information that is populated from the Party but you can add new contacts.

Add the Contact medium characteristics that can be used to contact the customer.

  1. Click Contacts .
  2. [Optional] Click the icons under Actions to edit the existing contacts.
    • Click the ellipsis (...) for options to Mark as Preferred, Edit Contact, or Delete Contact

      The Mark as Preferred option is only available when the contact is not marked as preferred.

  3. Click Add to add a new contact.
  4. Select Medium Type.

    Type of the contact medium, such as, email address, telephone number, or postal address.

  5. Select Contact Type.

    The type of contact, for example, phone number such as mobile, fixed home, or fixed office.

    Note: By default, if this is the first instance of this Contact Type, it is marked as Preferred.

    Any subsequent entries for this Contact Type are not marked as Preferred.

  6. Complete the relevant fields for the selected Contact Type.
  7. Click Save to save as draft but not to create a customer.
    Note: Click Cancel to delete this record, or save as draft so that you can return to this at a later time.

    The Submit button is disabled until the required sections are completed.

    The Submit button becomes active when all the required fields are completed and a summary screen is presented. You have the option to edit sections or click Confirm to create the Customer.

Credit Profile

This section is pre-populated from the Party selection but you can add another credit profile if required by using the following instructions.

Add the Credit profile.

  1. Click Credit Profile.
  2. Click Add or Add Credit Profile.
  3. Add Credit Agency Type.

    These are predefined levels:

    • Manual
    • External Agency
  4. Select Credit Agency Name.
    • If you select Experian or another external agency for the Credit Agency Name, you click Get Score to get the credit rating from an external credit rating agency and the fields are filled with relevant credit rating data.
    • For a manual credit rating, you must add:
      • Rating Reference (required)
      • Rating Score (required)
      • Add Valid from and Valid until dates (optional)
    • For no rating required, you don't need to add any credit rating data
  5. Add Valid from and Valid until dates.
  6. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Customer.
      • You can save, complete the remaining sections and then submit to create the Customer.
      • You can click Save > Submit to create the Customer.

        The save option only saves your current input and does not create a Customer.

    • Click Submit to create the Customer.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Customer.
    • Click Cancel to return to the Customer listing screen.

As the required section is completed, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Customer is created.
