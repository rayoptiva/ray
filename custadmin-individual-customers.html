Creating customers

Create a customer by associating this record with a party, contact, and other entities

Each customer is associated with a particular contact, and party which you can add or select through search when you create the customer.

From the Home > Customer page, click the button to open the Create Customer page.

Click on each tab to populate and save the relevant details:

  • Customer
  • Contacts
  • Credit Profile
  • Payment Method

Create a Customer

  1. Customer

    1. Select a Party

      To create a customer, you must assign a party by either of the following methods:
      • Select and click on an existing party through the Select Party search box.
        Note: A Party must be validated to be used for customer creation.
      • Click + Create Party to create a new party.
    2. Add the Valid from and Valid until dates.
    3. Click Save.
  2. Add Contacts
    1. Click the Contacts tab.
    2. Select Medium Type.
    3. Select a Contact Type.
    4. Add Phone Number.
    5. Add the Valid dates.
    6. Select Preffered if you want this option to be the preferred contact method.
    7. Click Save.

  3. Add Credit Profile

    1. Click Credit Profile.

    2. Add Credit Agency Type.
    3. Add Credit Agency Name
    4. Add ratings.
    5. Add the Valid dates.
    6. Click Save.

  4. Add Payment Method

    1. Click Payment Method.
    2. Select a Payment Method.
    3. Add bank account information.
    4. Add a Display Name.
    5. Select Preferred if you want this to be the preferred payment method.
    6. Click Save.
  5. Click Submit.

  6. View the summary and use the edit buttons to make any additional changes.

  7. Click Confirm to commit data to the database.

    Note: At any time during the data entry process for adding a new customer, you can click the Cancel button to delete the record, or save as draft to work on later.
