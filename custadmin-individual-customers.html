Creating customers

Create a customer by associating this reorod with a party, contact, and other entities

Create a customer as an individual or organization

To Create a customer [individual/organization]:

  1. Click the Create Customer icon on the left navigation panel.

  2. To create a customer, you must select an existing party through the search box or click Create to create the party.

  3. Select an Individual/Organization or create a new Party to use as the basis for the Customer.

  4. Select a contact to link to this customer by using the Choose Contact search box or create a new contact by completing the form.

  5. Click the Credit Profile button to add a customer credit profile and click Save.

  6. Click the Payment Method button to add a customer payment method and click Save.

  7. Click Submit and then Confirm to complete the customer creation process.
