Creating a Party

Create a Party (Individual) to use as basis for creating a customer

Create the Party (Individual) to define and record details of an individual (person) before using in a customer role.

When you create the customer, you select a Party to represent the individual (person) that is the person entity of the customer.

  • From the Dashboard click + Create > New Party
  • From the left navigation bar, click Party > + Create > Individual Party

Click the following sections in the right pane, add relevant details and click Save to save as draft or click Submit to create the party assuming that you've completed the required fileds denoted with an asterix (*)

  • Individual [Required]
  • Contacts
  • Individual Identification

    A party can't be validated until the Identification is verified.

    Note: Only a validated Party can be used to create a customer.
  • Credit Profile
  • Tax Exemption Certificate
  • Additional Information
    In the Additional Information section, you add the following specific information.
    • Language Ability
    • Skill
    • Disability
    • External Reference
    • Additional Information
    When you complete the required sections (Individual), the Submit button is enabled, which enables you to:
    1. Submit the data by clicking Submit.
    2. View the summary, use Edit and Update to make any changes.
    3. Confirm the data by clicking Confirm to create the Party.
Note: At any time during the data entry process you can click Cancel to Save as Draft or Delete to delete the record.

Individual

Add the Individual profile.

This is a required section where Given Name and Family Name are required fields.

  1. Complete the Basic Information and Personal Information sections.
  2. [Optional] Add another name as required.
  3. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can click Save, complete the remaining sections and then click Submit to create the Party.
      • You can click Save > Submit to create the Party when the required fields/sections are completed.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      Note: The Submit option is only available when all the required fields are completed.
      1. View the Party Summary and edit/update sections as required.
      2. Click Edit, make changes and then click Update to save changes.
      3. Click Confirm to confirm the data and the Party creation.
    • Click Cancel to return to the Party listing screen.
Note: As this is the only required section, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.

When you submit without validating by completing the Individual Identification section, the Party is in an Initialized state but not a Validated state.

Note: To use a Party for the creation of a customer, the Party must be validated.

Contacts

Add a contact for this Party.

Add the contact medium characteristics that can be used to contact a party.

  1. Click Contacts .
  2. Click + Add Contact or Add to add a new contact.
  3. Select Medium Type.

    The type of the contact medium, such as, email, telephone, or postal address.

  4. Select Contact Type.

    The type of contact depends on the Medium Type selection, for example, a medium type of telephone has contact types of mobile number or phone number, as shown in the following examples.

    • Email Medium Type selected in the previous step.
      • Options for Contact Type are:
        • Personal Email
        • Work Email
      • Email is a required field
      • Valid from and Valid until fields are optional
      • Select Preferred if you want this to be the preferred contact option.
      • Click Save
        Note: The first entry, for example, personal email is marked as Preferred by default. Subsequent entries, for example, work email, is not marked as Preferred by default but you can change this configuration after you save; under Actions, click ... > Mark as Preferred.
    • Telephone Medium Type selected in the previous step.
      • Options for Contact Type are:
        • Mobile Number
        • Phone Number
      • Number is a required field
      • Valid from and Valid until fields are optional
      • Select Preferred if you want this to be the preferred contact option.
      • Click Save
        Note: The first entry, for example, mobile number is marked as Preferred by default. Subsequent entries, for example, phone number, is not marked as Preferred by default but you can change this configuration after you save; under Actions, click ... > Mark as Preferred.
    • Postal Address Medium Type selected in the previous step.
      • Options for Contact Type are:
        • Head Office
        • Work
        • Home
      • Street, City, Post code, and State or Province are required fields.
      • Valid from and Valid until fields are optional
      • Select Preferred if you want this to be the preferred contact option.
      • Click Save
        Note: The first entry, for example, Work is marked as Preferred by default. Subsequent entries, for example, Home, is not marked as Preferred by default but you can change this configuration after you save; under Actions, click ... > Mark as Preferred.
  5. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then click Submit to create the Party.
      • You can click Save > Submit to create the Party when all the required fields/sections are completed.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      1. View the summary and edit/update sections if required.
      2. Click Edit, make changes and then click Update to save changes.
      3. Click Confirm to confirm the data and create the Party.

        You are not requied to click Save before you click Submit as the Submit action submits any data that is entered independently of the Save action, which saves your work locally.

    • Click Cancel to return to the Party listing screen.

As the required section (Individual) is completed, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.

Individual Identification

Add identification for the individual.

  1. Click Individual Identification.
  2. Click + Add Identification or Add.
  3. Select an Identification Type from the Identification Type menu.
    • Passport
    • National Identity Card
    • Drivers License
    • Social Security Number
    • Birth Certificate

      The following are required fields for all of the listed Identification Types.

      • Identification ID
      • Issuing Authority
      • Issuing Date
      • Upload File
  4. Select Validated to validate this party.
    • You can Mark as Validated at a later stage by selecting Mark as Validated under the Actions menu of the search results screen or when you edit the Party.
      Note: To be validated, a Party much have identification added that is validated.
  5. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then click Submit to create the Party.
      • You can click Save > Submit to create the Party.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Party.
    • Click Cancel to return to the Party listing screen.

As the required section (Individual) is completed, you can submit this record, review the summary, edit if required, and click Confirm to confirm the data that you've entered and the Party is created.

Credit Profile

Add the Credit profile.

  1. Click Credit Profile.
  2. Click Add or Add Credit Profile.
  3. Add Credit Agency Type.

    These are predefined levels:

    • Manual
    • External Agency
  4. Select Credit Agency Name.
    • If you select Experian or another external agency for the Credit Agency Name, you click Get Score to get the credit rating from an external credit rating agency and the fields are filled with relevant credit rating data.
    • For a manual credit rating, you must add:
      • Rating Reference (required)
      • Rating Score (required)
      • Add Valid from and Valid until dates (optional)
    • For no rating required, you don't need to add any credit rating data
  5. Add Valid from and Valid until dates.
  6. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then submit to create the Party.
      • You can click Save > Submit to create the Party.

        The save option only saves your current input and does not create a Party.

    • Click Submit to initiate the Party creation.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Party.
    • Click Cancel to return to the Party listing screen.

As the required section (Individual) is completed, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.
