Creating Subscriptions

There are several starting points to create a subscription in the user interface.

To create a subscription you create a product order.

  1. Select one of the following options to create a subscription by creating a product order.
    • From the Customer Dashboard on the Customer screen.
      1. Select a customer and click the eye icon to open the customer dashboard.
      2. Click the vertical Ellipsis (â‹®) and on the card and select Create New Order.
      3. Click Add Product on the Prepaid Activation dialog box.
      4. Click Confirm to go to the Create Product Order screen.
    • From the Account screen where you list accounts.
      1. Select an account and click the eye icon to open the account dashboard.
      2. Click Create New + on the Orders tab.
      3. Click Add Product on the Prepaid Activation dialog box.
      4. Click Confirm to go to the Create Product Order screen.
    • From the Dashboard
      1. Click + Create beside Orders.
      2. Click Select to select an account.

        If no accounts are listed, search by account number or phone number to list accounts.

      3. Select Add Product on the Prepaid Activation dialog box.
      4. Click Confirm to go to the Create Product Order screen.

  2. On the Create Product Order screen, select from Mobile | Fixed | Internet | IP TV under Select Product and Quanity.

    Only Mobile is available currently.

    • Select Mobile
      1. Select Mobile and click the + button to add one or more mobiles.

        By selecting two or more, you get the option to create in bulk, which makes the process faster.

      2. Under Select Type, select Prepaid, Postpaid or Hybrid and click Next.

        Only Prepaid is available currently

      3. Click Next.

      4. Choose a SIM type for the phone.

        • Select Physical SIM
          1. Enter an ICCID number and click Validate.
          2. Click Reserve to reserve a mobile number from your chosen category.

            The Reserve button will change to Reserved.

          3. Click Next.
          4. Click Select to select a Device or click Skip to bypass this section or click Next after you select phone to go to the next section.
          5. Select a Plan and any optional Add-ons that you want to add.
          6. Click Next.
          7. Select Contact Details or click + Create to create a new contact.

            Contact details are populated from Party/Customer records.

          8. Click Add to Cart.
        • Select e-SIM
          1. Click Reserve to reserve a mobile number from the chosen preferred category.
            Note: You can select a contact when you get to the contact screen so that the e-SIM QR code can be sent to the subscriber using the selected email.
          2. Click Next.
          3. Click Select to select a Device or click Skip to bypass this section or click Next after you select phone to go to the next section.
          4. Select a Plan and optional Add-ons.
          5. Click Next.
          6. Select Contact Details or click + Create to create a new contact.
          7. Select the email to send the QR Code Confirmation to, if you're presented with this option and then click Continue.
          8. Click Add to Cart.
  3. Click Go to Cart.
  4. Click Checkout to make payment or Add New Item to add another item.
  5. Select a payment method and then click Pay Now.
    • [Optional] You can edit or delete the address.
  6. Click Go to Account or Go to Home.
