Accounts

Create and manage customer accounts.

A customer account represents a financial entity which records all customer accounting events.

A customer is associated with one or more accounts.

An account can be:

  • Associated with an existing customer or an existing account
  • Created as a Subscriber account, Group account, or Group pool account
Note: Contacts for an account come from existing contacts in Party and Customer, which you select during account creation.

Get Accounts listing

You can search for existing accounts/customers using the search box on the Accounts page:
  • Type the search term in the search box.
    • Click the Advanced filter icon next to the search box to do an advanced search.
  • Use the filters to refine the returned search results:

    1. List
    2. Draft
    3. Dunned
    4. Suspended
  • Use the Actions menu to Edit or change status of selected accounts.
  • Click the eye icon () to view the account dashboard for a selected account.
  • Click the edit icon () to edit the selected account.
  • Click these icons to customize the search output display.

Create Accounts

To go to the Accounts creation page:
  • On the Accounts page, click the + Create button to open the Create Account page.
  • Click + Create > New Account on the customer management dashboard.
