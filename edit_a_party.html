Editing a Party

Edit or update a Party

  1. Navigate to the Party screen from the left navigation by clicking the icon or from any lookup/search and search for a specific party which you want to edit or update.
  2. Search for an existing Party by using the search box or the search icon to open an advanced search dialog.
  3. Filter your search results by using the filter icons which are represented as follows:

    1. = List all

    2. = Draft

    3. = Initialized

    4. = Validated

    5. = Deceased

  4. Customize the fields and order of the displayed results by clicking on the icons.
  5. Click the icons under Actions to make changes to a party.
    • Click the eye icon to view a specifc party and then click Edit to edit any section.
    • Click the pencil icon to edit a specific party.

      • Marking as validated if the party is not validated already.
      • Download a summary.
      • Delete the party.
  6. Make the desired changes in the fields and click the Submit button
  7. Once submitted, a summary page will be displayed to view the changes.
  8. Click the Confirm button to update the party data.
  9. [optional] Click the Export button to download as a PDF file.
