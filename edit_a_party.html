Editing a Party

Edit or update a Party

  1. Navigate to the Party screen from the left navigation by clicking the icon or from any lookup/search and search for a specific party that you want to edit or update.
  2. Search for an existing Party by using the search box or click the advanced search icon to use the advanced search dialog.
  3. Filter your search results by using the filter icons which are represented as follows:

    1. List all: Get a complete list of all records.

    2. Draft: Return records that were save as draft.

    3. Initialized: Return records that were submitted but not validated.

    4. Validated: Return records that are validated. Validated records must have the identification verified.

    5. Deceased: Return records of deceased parties.

  4. Click the icons to change layout and order of the visible records in the results pane.
  5. Click the icons under Actions to make changes to a party.
    • Click the eye icon to view a specifc party summary and edit any section by clicking Edit.
      1. Click the Update button if you make any changes.
      2. Click the Confirm button to update the party data.
    • Click the pencil icon to edit a specific party and click Edit to make changes.
      1. Click the Update button if you make any changes.
      2. Click the Confirm button to update the party data.
