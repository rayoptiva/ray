Create Account

Create an account based on an existing customer or an existing account.

Access the Create Account page by either of the following methods:
  • Click the Account icon on the left navigation bar and then click + Create
  • Click + Create > New Account on the customer management dashboard.
Note: Accounts are associated with existing customers or existing accounts.

To create an account, complete the following applicable sections:

Account

Create an account for a customer. This is a required section.

  1. On the Create Account page, search for Customers or Accounts in the Select Customer or Account search box.
  2. Select a Customer or an Account on which to base the account creation.

    The relevant customer or account information is loaded automatically.

    • Select a Billing Type.
      Note: The Prepaid option does not include tabs for Supplementary Data and Service Level Discounts.
    • Enter an Account Name.
    • Select an Account Role.
    • Select an Account Type.
  3. Select a Group Type:
    • Subscriber
    • Group
    • Group Pool
      If you select Group or Group Pool, select Responsible if this is a parent account and you want make it a responsible account for any child accounts that are created.
      Note: A maximum of five non-responsible child accounts are permitted for a responsible account.
  4. Select a Credit Category from the Credit Category menu.
  5. Select a Dealer Code from the Dealer Code menu.
  6. [Postpaid only] Select a tax rate from the Tax Authority menu.
    Note: The Tax Authority is not configurable for Prepaid on this page - inherits from the Party.
  7. [Optional] Add a greeting.
  8. Click Save.
Note: You can click the Cancel button to cancel and return to the Account listing screen,

Contacts

Add a contact for this account.
Note: The contacts that you add must be pre-existing contacts for the Party/Customer to use with this account.
  1. Click the Contacts tab on the Create Account page.
  2. Click Add, or + Add Contact.
  3. Click Select Entity, and select Party or Customer from which the contact is added.
  4. Select a Medium Type.

    The selection that you make here must exist for the Individual/Party and Customer entities based on the Medium Type.

  5. Select a Contact Type.

    The selection that you make here must exist for the Individual/Party and Customer entities based on the Contact Type.

  6. Enter a contact name in the Contact Name field.
  7. Complete other relevant fields.
    Note: Each contact should have name and valid phone number.
  8. [Optional] Select Preffered to make this contact option the preferred option if it's not already the preferred option.
  9. Click Save.
Note: Click Cancel to delete this record, or save as draft, so that you can return to this at a later time.

Security Questions

Add a security questions for this account to be used for verifying identity. This is a required section.

  1. Click the Add Security Questions tab.
  2. Select a question and type an answer.
  3. Add at least two more questions with answers
  4. Save, Submit or Cancel.
  • Click Save to save your current input as a draft, which does not create the account.
    • You can click Save > Submit to create the Customer.

      The save option only saves your current input and does not create a Customer.

  • Click Submit to create the account.
    • View the summary and edit/update sections if required.
    • Click Edit, make changes and then click Update to save changes.
    • Click Confirm to confirm the data and create the account.
  • Click Cancel to return to the Account listing screen.
