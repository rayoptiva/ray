Create Account

Create an account based on customer or existing account.

Access the Create Account page by either of the followind methods:
  • Click the Account icon on the left navigation bar and then click
  • Click > New Account on the customer management dashboard.
Note: Accounts are associated with existing customers or existing accounts.

To create an account, complete the following applicable sections:

Creating an Account

  1. On the Create Account page, search for Customers or Accounts.
  2. Select a Customer or an Account on which to base the account creation.
    • Select a Billing Type

      This Prepaid option does not include tabs for Supplementary Data and Service Level Discounts.

    • Enter an Account Name.
    • Select an Account Role.
  3. Complete other fields as required.
    Note: The Tax Authority is not configurable on this page.
    • If you select Group or Group Pool for Group Type, select Responsible if this is a parent account and you want make it a responsible account for any child accounts that are created.
      Note: A maximum of five non-responsible child accounts is permitted for a responsible account.
  4. Click Save.
Note: Click the Cancel button to delete the account, or keep as draft and the account creation is ceased.

Add Account Contact

Add a contact for this account.

  1. Click the Contacts tab on the Create Account page.
  2. Click Add or + Add Contact.
  3. Select a Medium Type.
  4. Complete other relevant fields.
    Note: Each contact should have name and valid phone number.
  5. [Optional] To make this Medium Type the preferred option for this account, you must select the Preferred checkbox.
  6. Click Save.
Note: Click Cancel to delete this record, or save in draft form so that you can return to this at a later time.

Add Security Questions

Add a security questions for this account to be used for verifying identity.

  1. Click the Add Security Questions tab.
  2. Select a question and type an answer.
  3. Add at least two more questions and answers.
  4. Click Save.
Note: Click Cancel to delete this record, or save in draft form so that you can return to this at a later time.

Add Payment Method

Add a payment method for this account.

Note: The default payment method is cash.
  1. Click the Payment Method tab.

    View existing payment methods or search for payment methods.

  2. Click Add to add a new payment method.
  3. Select a payment method from the Payment Method dropdown menu.
    1. Add details for the chosen payment method.
    2. Select Preferred if you want this to be the preferred payment method.
  4. Click Save.
Note: Click Cancel to delete this record, or save in draft form so that you can return to this at a later time.

Supplementary Data [Postpaid]

  1. Click the Supplementary Data tab.
  2. Click Add to add supplementary data.
  3. Add CSA Validation Code.
  4. Add CSA Device Serial Number
  5. Click Submit.

Service Level Discount [Postpaid]

Add a service-level discount for this account.

  1. Click Add.
  2. Add the service level discount.
  3. Click Submit.

Notes and Attachements

  1. Click the Notes and Attachement tab.
  2. Click Add or + Add Notes & Attachement
  3. Add notes and upload and supporting files.
  4. Click Save.
